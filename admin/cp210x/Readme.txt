This installer is required when a Davis weather station is installed with a direct USB datalogger with CP210x chip.
It installs the driver for communicating over a USB with the WeatherLink datalogger. This is for Windows 10 ONLY!

29/05/2018: Product version: 6.7.6.2130; File version: 10.1.3.2130 (x32 and x64)
